# Group Call Crasher

Discord Group call crasher, useful to troll :D
Built in py3 using discord.py

# Run the crasher
- git clone https://github.com/strivano/call-crasher
- cd call-crasher
- python3 main.py <token> <group_id> <times_to_send>
